Super Mario Bros. 3 on the NES was one of the first games I ever played in my childhood. So I decided to recreate it in a modern way, by making it using Java. Caesar Cipher

{{ subtitle }}

One of the most widely known ciphers, this Caesar cipher program was built using C++. It allows for encryption and decryption. Upon starting the program the user is prompted for the cipher key and a message to encrypt. The newly encrypted message is then outputted to the screen using the command prompt.
